## 🛠️ Quick Start

```bash
# 1. Clone
git clone https://github.com/Moussandou/Bingeki.git
cd Bingeki-V2

# 2. Install
npm install

# 3. Configure .env
cp .env.example .env
# (Add your Firebase keys)

# 4. Run
npm run dev
```
